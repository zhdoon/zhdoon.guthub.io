Безопасное встраивание скриптов из непроверенных источников

Метод базируется на том, что браузер не даст украсть данные скрипту если он загружен в рамках iframe содержимое которого находится на отдельном домене. В сочетании с iframe SANDBOX и правильно настроенными CSP политиками на вашем домене метод делает встраивание библиотек из третьих источников безопасным

Чат встроен c помощью inline iframe c атрибутами SANDBOX

 
<iframe src="https://upvote.ml/chat/index.html" sandbox="allow-same-origin allow-forms allow-scripts" ></iframe></pr>

Если нужно встроить чат с заданными параметрами - можно разместить JS инициализации чата на своем домене отличном от домена биржи и не на сабдомене, далее добавить адрес на страницу чата как в примере выше.

Например можно просто создать бесплатный html хостинг на github, создать index.html с инициализированным чатом, а далее вставить с помощью inline iframe с адресом репозитория блок на биржу
